# Prototipo HTML - People Management

Este prototipo representa a visao inicial do sistema descrito no PRD:
- Dashboard com cards de liderados.
- Tela individual com secoes estruturadas.
- Icone `i` com perguntas exploratorias (tooltip).
- Historico de alteracoes por secao.
- Bloco de radar (RF-13) com exemplo visual.

## Como abrir
1. Abra `index.html` no navegador.
2. No dashboard, clique em um card para ir para a visao individual.
3. Passe o mouse no icone `i` para ver perguntas exploratorias.

## Estrutura
- `index.html`: layout principal
- `styles.css`: estilo visual
- `app.js`: mock de dados e interacoes

## Observacao
E um prototipo estatico (sem persistencia local e sem backend), focado em validar UX e organizacao das informacoes.
